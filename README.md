# FUTURE_CS_02

## Phishing Email Detection & Awareness System

This repository contains Task 2 completed as part of the Future Interns Cyber Security Internship Program.

The objective of this project is to analyze phishing email samples, identify common phishing indicators, classify email threats, and create a professional awareness and prevention report.

---

## Objectives

- Identify phishing indicators in emails
- Analyze suspicious email headers and domains
- Detect spoofed senders and malicious links
- Classify emails as:
  - Safe
  - Suspicious
  - Phishing
- Provide security awareness recommendations
- Create a professional phishing awareness report
